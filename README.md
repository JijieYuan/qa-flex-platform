# QA Flex Platform

Standard separated project structure:

- `backend/`: Java 21 + Spring Boot
- `frontend/`: Vue 3 + Element Plus
- `tools/`: local Java and Maven toolchain

## Start backend

```powershell
cd backend
.\run-backend.ps1
```

Backend default URL:

- `http://localhost:18080/`

## Start frontend

```powershell
cd frontend
.\run-frontend.ps1
```

Frontend default URL:

- `http://localhost:18181/`
